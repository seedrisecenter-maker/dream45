## 2단계: 콘텐츠 폭격 (2~4주)

### YouTube Shorts 전략
| 일정 | 콘텐츠 | 목표 |
|------|--------|------|
| 매일 | 45초 명언 Shorts 1편 | 알고리즘 노출 |
| 매주 월 | 명언의 탄생 이야기 5~10분 | 깊이 있는 콘텐츠 |
| 매주 수 | 4개국어 명언 낭독 ASMR | 글로벌 시청자 |
| 매주 금 | 꿈 인터뷰 시리즈 | 커뮤니티 참여 |

### 크로스 플랫폼 동시 배포
1. **TikTok** — 45초 Shorts 동시 업로드 + 트렌드 음악
2. **Instagram Reels** — 감성 편집 버전
3. **Instagram 피드** — 명언 카드뉴스 (정사각형)
4. **Pinterest** — 명언 포스터 이미지 → 장기 SEO
5. **Twitter/X** — 매일 명언 1개 + 4개국어
6. **Facebook** — 명언 카드 + 커뮤니티 그룹 생성

## 3단계: 바이럴 챌린지 (4~8주)

### #Dream45Challenge

**콘셉트**: 고개를 45도 들고 자신의 꿈을 말하는 영상 촬영

**참여 방법**:
1. 고개를 45도 올리고 셀카/영상 촬영
2. "나의 꿈은 ___입니다" 한 줄 작성
3. #Dream45Challenge #꿈꾸는45도 해시태그
4. 친구 3명 태그하여 꿈 릴레이!

**확산 전략**:
- 한국 인플루언서 5~10명에게 챌린지 참여 요청
- 인도/중국/미국 인플루언서에게도 각 언어로 동시 확산
- 최고의 참여 영상은 메인 채널에서 리포스트

### #45SecondDream
- TikTok 전용 챌린지
- 45초 안에 자신의 꿈 이야기하기
- 배경음악: 감성적인 피아노 트랙

### 연예인 접근 무료 방법
1. **태그 전략**: 모든 SNS 포스팅에 관련 연예인 태그
2. **팬 커뮤니티 활용**: 팬카페/Reddit에 프로젝트 소개
3. **챌린지 참여 유도**: 셀럽이 참여하기 쉬운 45초 포맷
4. **인용 활용**: 셀럽의 기존 명언을 4개국어로 번역하여 크레딧 표기
5. **프레스킷 제작**: 셀럽 매니저에게 보낼 프로젝트 소개서

## 5단계: 커뮤니티 & 파트너십 (8주~)

### 교육기관 파트너십
- 한국: 학교 교실에 명언 포스터 기증 캠페인
- 인도: NGO와 협업하여 교육 프로그램에 명언 활용
- 글로벌: UNESCO, UNICEF 교육 프로그램 연계 제안

### 오프라인 확산
- 명언 스티커 무료 배포 (카페, 서점, 대학교)
- QR코드 포함 — 스캔하면 dream45.vercel.app으로
- 거리 인터뷰 촬영 — "당신의 꿈은 무엇인가요?"

### 글로벌 이벤트 연계
- 세계 꿈의 날 (World Dream Day, 9/25) 특별 캠페인
- UN 국제 청소년의 날 (8/12) 연계
- TEDx 강연 지원
