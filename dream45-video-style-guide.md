# 꿈 꾸는 45도 — 영상 스타일 가이드

**"세상 사람들이 다시 꿈을 꿀 수 있도록"**

---

## 우리만의 포지셔닝

> Be Inspired의 **감성 회복** + RedFrost의 **철학적 깊이** + Goalcast의 **실화 스토리텔링**
> 여기에 **4개국어(한/영/중/힌디)** 라는 독보적 차별점

### 경쟁 공백
한국어로 시네마틱 명언 영상을 하는 채널이 **없다**.
- 세바시 = 강연 기반
- 체인지그라운드 = 카드뉴스 기반
- "꿈 꾸는 45도" = **시네마틱 명언 + 동양 철학 + 4개국어** → 이 자리가 비어있다

## 2. Shorts 제작 공식 (45초 영감)

### 구조: 45초 = 3막

| 구간 | 시간 | 내용 | 비주얼 |
|------|------|------|--------|
| **훅** | 0~3초 | 충격 질문 or 역설 진술 | 패턴 인터럽트 (어둠→빛 전환) |
| **전개** | 3~35초 | 명언 + 배경 스토리 + 해설 | 시네마틱 B-Roll + 키네틱 타이포 |
| **클라이맥스** | 35~45초 | 핵심 한 줄 + CTA | 풀스크린 명언 + 금색 텍스트 |

### 7가지 훅 공식 (우리 버전)

1. **역설 질문**: "성공한 사람들은 왜 실패를 사랑할까?"
2. **자기 동일시**: "꿈을 포기하려는 당신에게"
3. **충격 사실**: "이 한 문장이 넬슨 만델라를 27년 감옥에서 버티게 했다"
4. **반직관 진술**: "노력하지 마세요" (뒤집기 → 명언 연결)
5. **카운트다운**: "인생을 바꾸는 3가지 명언"
6. **도전**: "이 영상을 끝까지 보면 당신은 달라질 겁니다"
7. **직접 약속**: "45초 후, 당신의 하루가 바뀝니다"

### 자연 루프 설계
- 마지막 프레임의 시각적 톤 = 첫 프레임과 연결
- 끝에 "다시 들어보세요"가 아닌, 자연스럽게 처음으로 이어지는 구조
- 강제 루프 트릭은 2026년 알고리즘이 감지 → 자연 루프만 유효

## 3. 비주얼 스타일

### 메인 스타일: "밤하늘 시네마틱"
- 어두운 배경 (밤하늘, 우주, 도시 야경)
- 빛의 대비 (별빛, 촛불, 일출)
- 느린 움직임 (타임랩스, 슬로모션 자연)
- 45도 각도의 시각적 모티프 반복

### 편집 리듬
- 매 1~2초마다 시각적 변화 (앵글, 줌, 장면)
- 5~8초마다 마이크로 오픈 루프 (새로운 시각적 긴장)
- 점프 컷 활용 (데드 에어 제거)
- 명언 클라이맥스와 음악 빌드업 동기화

### 자막 스타일: "다이내믹 미니멀리즘" (2026 트렌드)
- 깔끔한 산세리프 + 흰색 텍스트 + 은은한 그림자
- 한 번에 3~4단어만 표시
- 핵심 키워드만 금색(#FFD97D) 하이라이트
- 부드러운 페이드인/스케일업 애니메이션
- 화면 중앙 배치 (하단/우측은 플랫폼 UI에 가려짐)

## 7. 콘텐츠 템플릿 3종

### A. "오늘의 명언" (15~25초) — 매일
```
[0~2초] 텍스트 훅: "오늘 당신에게 필요한 한 마디"
[2~5초] 어둠에서 빛으로 전환 + 음악 시작
[5~18초] 명언 텍스트 키네틱 타이포 + B-Roll
[18~22초] 명언 전체 풀스크린 (4개국어 순차 표시)
[22~25초] 로고 + "내일도 꿈꿔요" + 구독 CTA
```

### B. "명언의 탄생" (40~60초) — 주 2회
```
[0~3초] 충격 훅: "이 한 문장 때문에 세상이 바뀌었다"
[3~15초] 인물 소개 + 시대 배경 (B-Roll)
[15~35초] 명언이 탄생한 순간의 스토리
[35~50초] 명언 풀스크린 + 4개국어
[50~60초] 교훈 요약 + 공유 CTA
```

### C. "4개국어 낭독" (30~45초) — 주 1회
```
[0~3초] 훅: "4개 언어로 듣는 [주제]"
[3~12초] 한국어 낭독 + 한글 자막
[12~21초] 영어 낭독 + 영문 자막
[21~30초] 중국어 낭독 + 중문 자막
[30~39초] 힌디어 낭독 + 힌디 자막
[39~45초] 4개국어 동시 표시 + 구독 CTA
```
