## 1. 전략 개요

### 브랜드 포지셔닝
"꿈 꾸는 45도"는 동서양 위인들의 명언을 한국어, English, 中文, हिन्दी 4개국어로 전달하는 영감(Inspiration) 브랜드이다. YouTube, Instagram, TikTok 등 멀티 플랫폼에서 운영하며, 웹사이트(dream45.vercel.app)와 굿즈 라인도 병행한다.

### 아웃리치 목표
- **1단계 (0-3개월)**: 마이크로 인플루언서 10명과 콜라보, 팔로워 5,000 달성
- **2단계 (3-6개월)**: 미드티어 인플루언서 5명 참여, 팔로워 20,000 달성
- **3단계 (6-12개월)**: 메가 인플루언서/셀럽 1-2명 참여, 바이럴 챌린지 론칭

### 핵심 원칙
1. **가치 정렬(Value Alignment)**: 단순 광고가 아닌, 꿈과 영감이라는 가치를 공유하는 인물과 협업
2. **다국어 우선(Multilingual First)**: 4개국어 콘텐츠라는 독특한 USP를 전면에 부각
3. **진정성(Authenticity)**: 셀럽이 직접 공감하는 명언을 선택하도록 유도
4. **상호 혜택(Mutual Benefit)**: 셀럽에게도 의미 있는 콘텐츠 참여 기회 제공

## 6. 명언/추천사(Endorsement) 확보 방법

### 6-1. 무료로 셀럽 추천사를 얻는 5가지 방법

#### 방법 1: 기존 공개 발언 활용 (가장 쉬움)

셀럽이 이미 공개적으로 한 말(인터뷰, SNS, 연설)을 정당하게 인용하는 방식이다. 별도의 허가 없이 출처만 명확히 밝히면 된다.

**실행 방법:**
- 셀럽의 인터뷰, TED Talk, SNS 게시물에서 꿈/도전/희망 관련 발언을 수집
- 4개국어로 번역하여 명언 카드/영상으로 제작
- 원본 출처를 명확히 표기 (예: "2005 Stanford Commencement Speech")
- 해당 셀럽을 태그하여 게시

**주의사항:**
- 상업적 사용 시 저작권/퍼블리시티권 확인 필요
- "추천"이 아닌 "인용"으로 명확히 구분
- 맥락을 왜곡하지 않도록 주의

#### 방법 2: Cameo를 통한 저비용 영상 추천사

Cameo 플랫폼을 활용하면 $3-$500 수준에서 셀럽의 개인화된 영상 메시지를 받을 수 있다.

**실행 방법:**
- Cameo.com에서 동기부여 카테고리 셀럽 검색
- $100 이하 가격대의 셀럽 1,600명+ 중 브랜드와 맞는 인물 선정
- 요청 메시지: "꿈 꾸는 45도 프로젝트에 응원 메시지를 부탁드립니다. 꿈을 향해 나아가는 전 세계 사람들에게 영감을 주는 한마디를 해주세요."
- 받은 영상을 SNS 콘텐츠로 활용 (Cameo Business 라이선스 확인 필요)

**비용 참고:**
- 일반 인플루언서: $25-$150
- 마이크로 셀럽: $150-$500
- 유명 셀럽: $500-$15,000

#### 방법 3: 제품 시딩 (Product Seeding)

무료 굿즈를 셀럽에게 발송하여 자발적 언급을 유도한다.

**실행 방법:**
- "꿈 꾸는 45도" 프리미엄 굿즈 세트를 셀럽에게 발송
  - 패키지 구성: 에코백 + 머그컵 + 포스터 + 엽서 세트 + 개인화된 손편지
  - 손편지에 해당 셀럽과 연관된 명언을 4개국어로 적어서 동봉
- 발송 시 SNS 언급을 직접 요청하지 않음 (부담을 주면 역효과)
- 대신 패키지 자체가 "공유하고 싶은" 수준의 품질과 감성을 갖추도록 제작
- 셀럽의 에이전트/매니저 주소로 발송 (공식 Fan Mail 주소 활용)

**팁:**
- 신예/떠오르는 셀럽(up-and-coming celebrities)에게 집중하면 성공률이 높음
- A-list 셀럽은 매주 수십 개의 제품을 받으므로, 차별화된 패키징이 핵심

#### 방법 4: 자선/사회공헌 연계

셀럽의 자선 활동이나 재단과 연계하여 접근한다.

**실행 방법:**
- 셀럽의 재단/자선 활동 조사 (예: RM의 교육 관련 기부, Oprah의 리더십 프로그램)
- "꿈 꾸는 45도" 굿즈 수익의 일부를 해당 재단에 기부하겠다는 제안
- 교육 소외 지역에 4개국어 명언 포스터를 무료 배포하는 프로그램과 연계
- 셀럽의 사회공헌 팀에 프로포절 발송

#### 방법 5: 팬 커뮤니티를 통한 간접 접근

팬 커뮤니티의 힘을 빌려 셀럽의 주목을 받는다.

**실행 방법:**
- 셀럽의 명언/발언을 4개국어로 번역한 고퀄리티 콘텐츠를 제작
- 팬 커뮤니티에 공유하여 자발적 확산 유도
- 팬들이 해당 콘텐츠를 셀럽에게 태그하도록 자연스럽게 유도
- 충분한 바이럴이 이루어지면 셀럽이 직접 반응할 가능성 증가

## 8. 바이럴 챌린지 아이디어

### Challenge 1: #Dream45Challenge (핵심 챌린지)

| 항목 | 내용 |
|------|------|
| **컨셉** | 45초 안에 자신의 꿈을 말하는 숏폼 영상 |
| **형식** | (1) 하늘을 45도 올려다보는 장면으로 시작 (2) "나의 꿈은 ___입니다" (3) 왜 그 꿈을 꾸는지 한 문장으로 설명 (4) "당신의 꿈은 무엇인가요?" 로 마무리 (다음 참여자 지명) |
| **음악** | 오리지널 사운드 제작 (잔잔한 피아노 + "Dream at 45 degrees" 위스퍼) |
| **해시태그** | #Dream45Challenge #꿈꾸는45도 #45SecondDream |
| **참여 유도** | 3명 지명 릴레이 방식 (Ice Bucket Challenge 형태) |
| **인센티브** | 월간 Best Dream 선정, "꿈 꾸는 45도" 굿즈 증정. 누적 참여 1만 건 달성 시 대형 이벤트 |
| **셀럽 참여 유도** | Phase 1: 마이크로 인플루언서 10명이 먼저 시작 -> Phase 2: 미드티어 인플루언서 태그 -> Phase 3: 메가 셀럽 자연스럽게 유입 |

### Challenge 2: #4Languages1Dream (다국어 특화)

| 항목 | 내용 |
|------|------|
| **컨셉** | 같은 명언을 4개국어로 말하는 릴레이 챌린지 |
| **형식** | (1) 1인이 4개국어 중 1개로 명언 낭독 (2) 다음 참여자를 태그하여 다른 언어로 이어가기 (3) 4명이 모이면 1개 명언 완성 |
| **해시태그** | #4Languages1Dream #OneQuoteFourLanguages |
| **바이럴 포인트** | 다국어 능력을 자랑하고 싶은 사람들의 참여 욕구 자극. 각국 언어 학습자 커뮤니티와 자연스럽게 연결 |
| **셀럽 연결** | 다국어 가능 셀럽에게 특히 매력적 (Jackson Wang, RM 등) |

### Challenge 3: #LookUp45 (비주얼 챌린지)

| 항목 | 내용 |
|------|------|
| **컨셉** | 45도 각도로 하늘을 올려다보는 아름다운 사진/영상 공유 |
| **형식** | (1) 자신만의 장소에서 45도 하늘 사진/영상 촬영 (2) 그 순간 떠오른 꿈이나 명언을 캡션으로 작성 (3) 위치 태그 포함 |
| **해시태그** | #LookUp45 #45DegreesSky #하늘45도 |
| **바이럴 포인트** | 전 세계 다양한 장소의 45도 하늘 사진이 모이는 글로벌 갤러리 효과. 여행/풍경 크리에이터와 자연스럽게 연결 |
| **확장** | 최종적으로 전 세계 #LookUp45 사진을 모아 모자이크 아트 또는 온라인 전시회 개최 |

### Challenge 4: #BeforeAndAfterDream (변화 챌린지)

| 항목 | 내용 |

| **음악** | 잔잔한 피아노에서 웅장한 오케스트라로 전환 |
| **해시태그** | #BeforeAndAfterDream #DreamTransformation #꿈의전환 |
| **바이럴 포인트** | Transformation 콘텐츠는 TikTok에서 600M+ 조회수 기록 트렌드. 감정적 공감 유발 |

### Challenge 5: #QuoteRelay45 (명언 릴레이)

| 항목 | 내용 |
|------|------|
| **컨셉** | 자신에게 가장 의미 있는 명언 1개를 공유하고, 다음 3명을 지명하는 릴레이 |
| **형식** | (1) 화면에 명언 텍스트 등장 (2) 참여자가 그 명언과 자신의 이야기를 30초 내외로 공유 (3) "이 명언을 [이름1], [이름2], [이름3]에게 전합니다" |
| **해시태그** | #QuoteRelay45 #PassTheQuote #명언릴레이 |
| **셀럽 참여 유도** | 셀럽들이 자신의 인생 명언을 공유하는 것은 부담이 적으면서도 의미 있는 참여. 팬들이 자발적으로 셀럽을 지명 |

| **형식** | (1) 45초 타이머 시작 (2) 하늘을 바라보며 명상 (3) 그 45초 후 느낀 감정/생각을 한 문장으로 공유 (4) 관련 명언 1개 인용 |
